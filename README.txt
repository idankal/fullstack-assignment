# Start Project

1. Need to have "yarn" install.
2. "npm install -g yarn".

3. Open cmd in root foler path, install packages with the command: "yarn install".
4. Run "node server.js" command, to open server.

5. Open another cmd, go to client folder path, run command "yarn install".
6. Run command "yarn start" in client folder.

7. Open browser url: "http://localhost:4000/" for assignment task
8. Open browser url: "http://localhost:3000/" for react client
